# Code of Conduct

`juvantlabs` adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

The full text is available at the link above. Read it before contributing.

## Reporting

Report concerns about contributor behavior to:

- Email: [dev@juvant.io](mailto:dev@juvant.io)
- Subject prefix: `[CoC]`

Reports go to the org maintainers and are handled per the Contributor
Covenant's enforcement guidelines (see the linked text). Confidentiality is
respected for both reporter and reported party.

## Scope

This Code applies to all `juvantlabs/*` repositories — issues, pull requests,
discussions, code review, and any other interaction in the org's GitHub
spaces. Off-platform conduct that affects on-platform participation (e.g.
harassment via email, social media targeting an org member as a maintainer)
is also in scope.
